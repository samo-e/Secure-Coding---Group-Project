# Communication and Responsibilities

## Communication

primarily chat on discord for progress updates and planning.

organize calls as needed.

**Progress Tracking**: github issues

**Accountability**: Assign yourself to the issue on github related to whatever you choose to work on.

# Version Control Strategy

## Hosting

project will be hosted on github

## Contributions Strategy

Create/assign yourself to the github issue before starting.

Work on feature branches and create a pr once done.

## Risk Management & QA

github issues must detail what you are working on and what you plan to do in future so that another member can continue your work in case you are unavailable in future for any reason.

Wait at least one of the group members has approved your pr before merging to ensure code quality
